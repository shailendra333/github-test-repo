## Translation Methods

This system provides **4 translation methods** that can be compared side-by-side:

1. **Azure Translator** - Microsoft's fast, reliable translation service
2. **Azure OpenAI** - Context-aware AI translation with custom rules
3. **CrewAI Agents** - Multi-agent collaborative translation (3 agents)
4. **Hybrid (Azure + CrewAI)** - Optimized combination (Azure + 2 agents)
